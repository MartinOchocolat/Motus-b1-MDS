Bienvenue dans le motus de Martin, spécial personnages de jeux vidéos.
Chaque fois que la page est lancée, un nom aléatoire de 5 lettres sera choisi parmi une liste, à vous de le retrouver !

Pour ce faire, entrez 5 lettres et validez.
S'afficheront en rouge les lettres absentes, en vert les lettres à la bonne position et en orange les lettres qui sont présentes dans le mot mais mal placées.

Vous avez 6 essais pour trouver le mot. Si vous le trouvez, bien joué le jeu s'arrête et les espaces vides de la grille disparaissent. Sinon, tant pis le mot à trouver s'affiche alors.

Un bouton reset se trouve en haut à droite si vous avez envie de restart.
